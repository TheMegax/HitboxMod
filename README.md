## Hitbox Mod
![Requires BTD6 Mod Helper](https://i.imgur.com/ID7iwfY.png)
This mod adds toggleable hitboxes to the game. This includes tower collision, projectiles, bloons, ~~map obstacles~~ (soon), and paths.
